# LinkDock
Простой менеджер ссылок для Windows 10 (Electron).

## Быстрый старт
```powershell
npm i
npm start
```

## Сборка установщика
```powershell
npm run build
```
Готовый файл: `dist/LinkDock-Setup-x.y.z.exe`

> Для автообновлений поменяйте `build.publish[0].url` в `package.json` на ваш хостинг.
